# Flutter 에서 고정된(static) image 사용하기

- 프로젝트 폴더에 이미지를 저장할 폴더 만들기
- 생성한 폴더에 사용할 이미지를 복사 하여 저장하기
- 이미지 이름은 snack case 로 사용해야한다 (소문자에 `_` 사용 )
